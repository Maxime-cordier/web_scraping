This procedure is not considered as application of multiple line-ups on the same runway as defined by Doc 7030,Part 3, paragraph 3.1.1 which reads that “Line-ups instructions may be issued to more than one aircraft at different points on the same runway …”. ANSPs which are using multiple line-ups from the same runway access point shall consider this procedure as an application of a conditional ATC clearance for sequencing of departing traffic.
