Recovery, Analysis & Presentation System (RAPS) provides a ground data replay and analysis station including flight animation for FOQA programs as well as aircraft accident and incident investigations. Recovery, Analysis & Presentation System (RAPS) and Insight provide decoding, analysis, and animation tools that can be used for the investigation of data originating primarily from Flight Data Recorders (FDRs) and Cockpit Voice Recorders (CVRs), as well as secondary sources such as radar and Global Positioning System (GPS). Recovery, Analysis & Presentation System (RAPS) and Insight primary strengths are tools that allow the flight data analyst to quickly focus on pertinent information. Analysis of flight data using Recovery, Analysis & Presentation System (RAPS) is a primary source of information for occurrence investigation as well as routine data monitoring to identify trends and study exceedances. Recovery, Analysis & Presentation System (RAPS) was developed by company Flightscape.
