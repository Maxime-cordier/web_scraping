Analysis of maintenance error data collected by a group of UK Maintenance Organisations found that when the type of error was classified, four categories accounted for 78% of the errors. These were Installation error - 39%, Inattention (damage) - 16%, Poor inspection standards - 12% and Approved data not followed - 11%. The presentation of this data was accompanied by some solutions for both ‘people’ and ‘process’ for all the main types of error found.
