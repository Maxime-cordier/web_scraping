Where controllers choose to query the discrepancy, the phraseology to be used is: “(Callsign), Check Altimeter Setting, QNH xxx” - where xxx is the local QNH being utilised Notwithstanding the above, if an aircraft is in conflict with another, the priority is to resolve the confliction. If appropriate, avoiding action should be issued prior to confirming the correct QNH. Where sufficient time exists, the use of the following phraseology may be used to query the altimeter setting used by the aircrew: “(Callsign) Report QNH” This will enable flight crew to check selected pressure setting, irrespective of any BAT Advisory or QNH data displayed in the Mode S Data Window.
