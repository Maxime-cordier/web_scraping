The ADS-B data transmitted are defined in the relevant standards and certification documents (e.g. EASA AMC 20-24 for ADS-B in Non-Radar Airspace or CS-ACNS for “ADS-B out”). They include (amongst others) the following: Aircraft horizontal position (latitude/longitude). Aircraft barometric altitude (will be the same as for the SSR). Quality indicators. Aircraft identification: Unique 24-bit aircraft address Aircraft identification Mode A code (in the case of CS ACNS for “ADS-B Out”). Unique 24-bit aircraft address. Aircraft identification. Mode A code (in the case of CS ACNS for “ADS-B Out”). Emergency status. SPI (special position indicator) when selected.
